<h1>LZW-Encoder by Ruzal</h1>
<h2>Как запустить?</h2>

```
g++ TX/TXLib.h Dictionary.cpp Dictionary.h LZW.cpp LZW.h main.cpp -o lzw
```

Для сжатия пишем:
```
./lzw.exe 1 source.txt output.txt
```

Для распаковки пишем:
```
./lzw.exe 2 source.txt output.txt
```
